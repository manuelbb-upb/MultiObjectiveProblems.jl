MultiObjectiveProblems.MHT1 — Type
MHT1()

Bi-Objective, bi-variate problem MHT1 (T1) from Thomann, 2019, “Doctoral Thesis: A Trust Region Approach for Multi-Objective Heterogenous Optimization”.

Unconstrained, convex objectives, convex Pareto Front. $f₁$ expensive, $f₂$ cheap.

The objectives are given as

\[\begin{aligned} f_1( \mathbf{x} ) &= \frac{1}{2}x_1^2 + x_2^2 - 10 x_1 - 100,\\ f_2( \mathbf{x} ) &= x_1^2 + \frac{1}{2}x_2^2 - 10 x_2 - 100. \end{aligned}\]

MultiObjectiveProblems.MHT2 — Type
MHT2()

Bi-Objective, bi-variate problem MHT2 (T2) from Thomann, 2019, “Doctoral Thesis: A Trust Region Approach for Multi-Objective Heterogenous Optimization”.

Unconstrained, non-convex objectives, non-convex Pareto Front. $f₁$ cheap, $f₂$ expensive.

The objectives are given as

\[\begin{aligned} f_1( \mathbf{x} ) &= \sin( x₂ ),\\ f_2( \mathbf{x} ) &= 1 - \exp \left( - \left( x₁ - \frac{1}{\sqrt{2}} \right)^2 - \left( x₂ - \frac{1}{\sqrt{2}} \right)^2 \right) \end{aligned}\]

MultiObjectiveProblems.MHT3 — Type
MHT3()

Bi-Objective, bi-variate problem MHT3 (T3) from Thomann, 2019, “Doctoral Thesis: A Trust Region Approach for Multi-Objective Heterogenous Optimization”.

Box constraints [-2,2]², convex objectives, convex Pareto Front. $f₁$ cheap, $f₂$ expensive.

The objectives are given as

\[\begin{aligned} f_1( \mathbf{x} ) &= x₁ + 2,\\ f_2( \mathbf{x} ) &= x₁ - 2 + x₂. \end{aligned}\]

MultiObjectiveProblems.MHT4 — Type
MHT4(n)

Bi-Objective, n-variate problem MHT4 (T4) from Thomann, 2019, “Doctoral Thesis: A Trust Region Approach for Multi-Objective Heterogenous Optimization”.

Box constraints [-10,10]ⁿ, convex objectives, convex Pareto Front. $f₁$ expensive, $f₂$ cheap.

The objectives are given as

\[\begin{aligned} f_1( \mathbf{x} ) &= Σ_{i=1}^{n-1} xᵢ² + 2, \\ f_2( \mathbf{x} ) &= Σ_{i=1}^{n} xᵢ - 2 \end{aligned}\]

MultiObjectiveProblems.MHT5 — Type
MHT5()

Bi-Objective, bi-variate problem MHT5 (T5) from Thomann, 2019, “Doctoral Thesis: A Trust Region Approach for Multi-Objective Heterogenous Optimization”.

Box constraints [1e-12,30] × [0,30], convex objectives, convex Pareto Front.

$f₁$ expensive, $f₂$ cheap.

The objectives are given as

\[\begin{aligned} f_1( \mathbf{x} ) &= x₁ \ln( x₁ ) + x₂²,\\ f_2( \mathbf{x} ) &= x₁² + x₂⁴. \end{aligned}\]

MultiObjectiveProblems.MHT6 — Type
MHT6()

Bi-Objective, bi-variate problem MHT6 (T6) from Thomann, 2019, “Doctoral Thesis: A Trust Region Approach for Multi-Objective Heterogenous Optimization”.

Box constraints [1e-12,100]², convex objectives, convex Pareto Front.

$f₁$ expensive, $f₂$ cheap.

The objectives are given as

\[\begin{aligned} f_1( \mathbf{x} ) &= -\ln( x₁ ) - \ln( x₂ ),\\ f_2( \mathbf{x} ) &= x₁² + x₂. \end{aligned}\]

MHT7()

Bi-Objective, tri-variate problem MHT7 (T7) from Thomann, 2019, “Doctoral Thesis: A Trust Region Approach for Multi-Objective Heterogenous Optimization”.

Box constraints [0, 30]^3 convex objectives, convex Pareto Front. 9 $f₁$ expensive, $f₂$ cheap.

The objectives are given as

\[\begin{aligned} f_1( \mathbf{x} ) &= Σᵢ xᵢ⁴ + Σᵢ xᵢ³,\\ f_2( \mathbf{x} ) &= Σᵢ xᵢ \end{aligned}\]
